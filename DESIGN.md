## 0. 先说清楚："试卷复用"到底是什么意思

你原话的意思（已澄清）：

> 将来给我的 Excel 文件名可能**一模一样**，只是给的是**不同批次的新人**考的。比如 3 月一批、4 月一批，都叫 "Presale Frame Website Promotion"。

**系统现在就能处理，不用改结构：**
- 每个 Excel 文件 → 导入成 1 个"批次"（`exam_session`）。
- `exam_name`（考试名）取文件名前缀。
- 两个同名文件 → **同一个 exam_name + 两个批次**（batch/日期不同）。
- 所以"复用"= 同名考试多次导入，天然就是复用。

**唯一要注意的一点：** 给题目打"知识维度"标签时，按 `exam_name` 打一次，所有同名批次自动复用这份标签，不用每批重新打。

> 一句话总结：**你照常给我同名文件，我照常导，系统自动把它们归到同一个考试下、当成不同批次。** 不用你额外做什么。

## 1. 三大需求拆解

### 1.1 同名考试复用（不同批次新人）
- 见第 0 节。结构已支持，重点是"维度标签按 exam_name 复用"。

### 1.2 新人转正后的周期考试
- 新人转正后，考试节奏会变：月度考、季度考等。
- 给每次考试加一个**类型标签**：`onboarding`（入职认证）/ `monthly`（月度）/ `quarterly`（季度）。
- 给客服加一个**阶段**：新人 / 转正后。
- 类型标签和"复用"是正交的两件事，分开记就行（复用是同名，类型是节奏）。

### 1.3 客服自助看错题 + 知识维度推荐（大框架）
分三层：
1. **管理员录入（底座）**：把"每道题属于哪个知识维度"告诉系统。
   - 先定义维度字典，如：产品知识 / 服务流程 / 系统操作 / 售后政策 / 信用规则。
   - 再把每场考试的 Q1..QN 逐题绑定到维度。
2. **维度分析引擎**：
   - 这场考试里各知识维度占多少题（卷的结构）。
   - 这个客服在哪些维度错得最多（错题按维度聚合）。
3. **客服自助视图**：
   - 客服登录后看自己这场**哪些题被扣分**。
   - 系统推荐："你在『售后政策』错得最多，建议重点学这部分"。

## 2. 数据模型（精简版）

**保留现有表，只加 3 张轻量表 + 给现有表补几个字段：**

| 表 | 变化 | 说明 |
|----|------|------|
| `cs_reps` | 补 `stage`(新人/转正)、`login_id` | 阶段 + 自助登录标识 |
| `exam_sessions` | 补 `exam_type`(onboarding/monthly/quarterly) | 每场考试打类型标签。**不拆表**，复用靠 exam_name |
| `exam_results` | 不变 | 已存每题得分（subjects: Q1..QN），配合维度映射即可算错题 |
| `knowledge_dimensions`（新） | — | 维度字典（中英双语）：名称 + 描述 |
| `exam_question_dimensions`（新） | — | **题→维度映射**，按 `exam_name + Q编号` 存，同名批次自动复用 |
| `accounts`（新，可选） | — | 登录体系：rep_id / login_id / 密码 / 角色(admin/csr) |

**不需要的（已砍掉）：** `exam_papers` 试卷表、`questions` 题目表（题面）——它们是为"显示题目原文"服务的，可后续可选补，不影响维度分析和推荐。

### 维度分析怎么算
- **一场考试的维度占比** = 该 exam_name 各维度题数 / 总题数。
- **某客服的维度错题** = 他成绩里某题得分 < 该题满分，按 `exam_question_dimensions` 归到维度。
- **学习推荐** = 维度错题数降序，取 Top-N："建议重点学习：维度X（错M题）、维度Y（错K题）"。

## 3. 页面规划

### 管理员端
| 模块 | 功能 |
|------|------|
| 考试管理 | 现有三视图 + 按类型(入职/月度/季度)筛选；批次显示"维度占比" |
| 知识维度字典 | 维度增删改（中英双语） |
| 题目→维度录入 | 给每个 exam_name 的 Q1..QN 绑定维度（逐题或批量模板） |
| 周期/阶段 | 给考试打类型、给客服标阶段 |

### 客服自助端（新增）
| 页面 | 功能 |
|------|------|
| 登录 | 用工号或 Name 登录（凭证方案待定） |
| 我的考试 | 只看自己的历次记录 |
| 我的错题 | 某场哪些题被扣分 |
| 我的推荐 | 维度聚合 + 学习建议 |

**权限边界**：客服只能看自己的；管理员看全部。
